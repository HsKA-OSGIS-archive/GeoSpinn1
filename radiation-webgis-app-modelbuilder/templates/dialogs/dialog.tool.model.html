Creates a new layer
Available features are buffer, intersect, difference, combine.
Need to select the model, based on selection possible layers will appear. After choosing the layer/layers user can hit create button to create the model.
Distance for the buffer can be given otherwise model will create using default distance.
User guidelines to create JSON file :
 
             1) Type of functionality must be a string eg. buffer, intersect...
             2) The id must be an integer eg. 1, 2.
            
parameters :

                1) Layer name must be string
                2) Buffer distance must be integer
Json file example
                    "model_1" : [
                        {
                         "type": "buffer",
                         "id"  : 1,
                         "parameters" :{
                              "l1": "input_layer1",
                              "radius": {
                                "default_dist" : 200,
                                "user_defineable" : false,
                                "units" : "kilometres",
                                "dissolve" : true
                              }
                         }
                        }
                    ]
